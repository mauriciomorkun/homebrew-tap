# homebrew-tap

Homebrew tap for [Mauricio Morkun](https://mauriciomorkun.com)'s macOS tools.

## AudioRouterNow

Free, open-source macOS audio routing — send system audio to multiple outputs simultaneously.

```sh
brew install --cask mauriciomorkun/tap/audiorouternow
```

→ [audiorouternow.mauriciomorkun.com](https://audiorouternow.mauriciomorkun.com)
